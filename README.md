# GameSave Hub

GameSave Hub is a desktop application for managing game save data in internet cafes, gaming centers, and diskless PC environments.

It is designed for warnet or internet cafe setups where many players use shared client PCs, and where diskless systems may reset local files after restart.

The server manages users, game save locations, save data storage, and client access. Each player can use their own account so their game progress stays separate, centralized, and available when they move to another PC.

The server can start automatically when the app opens and can stay active in the tray while running in the background.

It can also open automatically when the PC starts. Updates can be installed from inside the server app with one click, using the configured update source.

The main purpose of GameSave Hub is to make game save handling easier, cleaner, and more consistent across multiple cafe PCs without requiring players or operators to manually save or load progress.
